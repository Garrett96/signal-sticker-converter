# signal-sticker-converter
Fast script to convert img(s) into the signal.org sticker format

The signal sticker ecosystem is under-served compared to competing similar products. This is a way to quickly and easily make contribution.

Create an `input` folder then dump any img set in. `./run` the script, take the `output` folder contents straight to Signal's sticker studio.

Upload Guide: https://support.signal.org/hc/en-us/articles/360031836512-Stickers
